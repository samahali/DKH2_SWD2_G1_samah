Div is block element but when give body height 100% for example and give this body margin: auto it will not center the Div in left and right but when give the div itself margin: auto div will be centered in x but it will not be centered in the y co-ordinate body its height will be the height of div. So margin works only in horizontal not vertical to make it works in vertical we need to use: 
            body {
                display: flex;
                justify-content: center;
                align-items: center;
            }
